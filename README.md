# 长坤机械订单管理系统

## 系统简介

长坤机械订单管理系统是一个专业的企业级订单管理解决方案，采用科技风炫酷深色主题设计，提供完整的订单生命周期管理功能。

## 功能特性

### 核心功能
- **用户认证与权限管理**：支持多用户登录，精细化权限控制
- **订单全生命周期管理**：从创建到交付的完整流程管理
- **实时数据监控**：仪表盘展示关键业务指标
- **多维度数据分析**：支持按客户、类别、状态等维度分析
- **系统配置管理**：灵活的字段配置和类别管理
- **数据备份恢复**：保障数据安全

### 技术特色
- **现代化UI设计**：科技风炫酷深色主题，响应式布局
- **实时数据刷新**：5秒自动刷新，保证数据实时性
- **丰富的可视化**：ECharts图表展示，直观的数据分析
- **动画效果**：Anime.js驱动的流畅交互体验
- **安全可靠**：JWT认证，密码加密存储

## 系统架构

### 前端技术栈
- **HTML5 + CSS3**：现代化网页标准
- **Tailwind CSS**：实用优先的CSS框架
- **JavaScript ES6+**：现代JavaScript特性
- **ECharts.js**：专业数据可视化
- **Anime.js**：轻量级动画库
- **PIXI.js**：2D渲染引擎

### 后端技术栈
- **Node.js**：JavaScript运行环境
- **Express.js**：Web应用框架
- **MySQL**：关系型数据库
- **JWT**：JSON Web Token认证
- **bcrypt**：密码加密

## 安装部署

### 环境要求
- **Node.js**：14.0或更高版本
- **MySQL**：5.7或更高版本
- **操作系统**：Windows/Linux/macOS

### 安装步骤

1. **克隆项目**
   ```bash
   git clone [项目地址]
   cd changkun-order-management-system
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **配置数据库**
   - 确保MySQL服务正在运行
   - 系统会自动初始化数据库和表结构
   - 默认管理员账号：谢军强 / Xie750021

4. **启动服务器**
   ```bash
   npm start
   ```
   或使用开发模式：
   ```bash
   npm run dev
   ```

5. **访问系统**
   - 前端地址：http://192.168.10.22:3001
   - API地址：http://192.168.10.22:3001/api

## 故障排除

### 常见问题

1. **数据库连接失败**
   - 检查MySQL服务状态
   - 验证数据库配置信息
   - 确保网络连接正常

2. **登录失败**
   - 检查用户名密码
   - 确认用户状态为active
   - 查看服务器日志

3. **页面无法访问**
   - 检查服务器是否启动
   - 验证端口号是否正确
   - 检查防火墙设置
