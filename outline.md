# 长坤机械订单管理系统 - 项目大纲

## 文件结构
```
/mnt/okcomputer/output/
├── index.html              # 登录页面
├── dashboard.html          # 仪表盘页面
├── orders.html             # 订单管理页面
├── analytics.html          # 数据分析页面
├── settings.html           # 系统设置页面
├── main.js                 # 前端主要JavaScript
├── server.js               # 服务器端代码
└── resources/              # 资源文件夹
    ├── bg.jpg             # 背景图片
    └── logo.png           # 公司Logo
```

## 页面功能

### 1. 登录页面 (index.html)
- 科技风炫酷深色主题
- 用户名和密码输入
- 数据库连接状态显示
- 安全登录验证

### 2. 仪表盘页面 (dashboard.html)
- 最近7天待交货订单
- 返修订单高亮显示
- 订单状态停留时间统计
- 订单趋势图表
- 统计卡片（总订单数、待交货、返修）

### 3. 订单管理页面 (orders.html)
- 订单列表展示
- 新建、编辑、删除订单
- 高级搜索和筛选
- 打印功能
- 数据导入导出

### 4. 数据分析页面 (analytics.html)
- 多维度数据分析
- 图表展示（柱状图、折线图、饼图）
- 自定义分析内容
- 导出分析结果

### 5. 系统设置页面 (settings.html)
- 用户管理
- 订单字段管理
- 订单类别管理
- 回收站管理
- 数据备份与恢复

## 核心功能模块

### 前端功能
- 响应式设计
- 实时数据刷新（5秒间隔）
- 数据可视化
- 表单验证
- 权限控制

### 后端功能
- MySQL数据库连接
- RESTful API
- 用户认证
- 数据CRUD操作
- 数据备份恢复
